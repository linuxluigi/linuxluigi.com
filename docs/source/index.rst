Linuxluigi.com Website's documentation!
=======================================

.. toctree::
   :maxdepth: 2
   :caption: Contents:

Documentation for the new `linuxluigi.com`_ which is not online right now. The sourcecode is hosted on
`github.com/linuxluigi/linuxluigi.com`_ and the website will be later deploy on `heroku.com`_.

The new version of `linuxluigi.com`_ will be completely rewritten and completely open source. Backend is still the
framework `Django`_  with `Wagtail`_ as CMS. The frontend will be `Angular 4`_. The purpose of the new version of
`linuxluigi.com`_ is to create a small it knowledge base for myself and add a portfolio function.

.. _github.com/linuxluigi/linuxluigi.com: https://github.com/linuxluigi/linuxluigi.com
.. _linuxluigi.com: https://linuxluigi.com
.. _heroku.com: https://heroku.com
.. _Django: https://www.djangoproject.com/
.. _Wagtail: https://wagtail.io/
.. _Angular 4: https://angular.io/
